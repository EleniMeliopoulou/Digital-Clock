# Digital Clock

This is a simple digital clock built with React + Vite.

---

## Technologies Used
* *Front-End:*
    * React with Vite 
    * CSS 

---

## Getting Started

Follow the steps below to set up and run the application locally.

Clone the Repository

```bash
git clone https://github.com/EleniMeliopoulou/Digital-Clock.git
cd Digital-Clock
```

Install Dependencies

```bash
npm install
```

Start the Front-End
```bash
npm run dev
```
The application will be available at:

http://localhost:5173
